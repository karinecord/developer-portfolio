## 5. Estruturas de Dados e Algoritmos

> **Tags:** #algoritmos #estrutura-de-dados #ordenacao #busca #arvore

### ✅ O que a Cesgranrio cobra
- [ ] **Busca sequencial**: O(n) — percorre elemento a elemento
- [ ] **Busca binária**: O(log n) — exige array ordenado
- [ ] **Bubble Sort**: compara pares adjacentes, O(n²)
- [ ] **Selection Sort**: seleciona o menor e coloca na posição, O(n²)
- [ ] **Insertion Sort**: insere cada elemento na posição correta, O(n²)
- [ ] **Lista encadeada**: nós com ponteiro para próximo, inserção/remoção O(1)
- [ ] **Pilha (Stack)**: LIFO — Last In, First Out
- [ ] **Fila (Queue)**: FIFO — First In, First Out
- [ ] **Árvore binária**: nó raiz, filhos esquerdo/direito, percursos (pré, in, pós-ordem)
